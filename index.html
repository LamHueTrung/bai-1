                        input: 4

Lý Thuyết

Gợi ý: Cần sử dụng: printf, scanf 
                        
int main() {
    int n; // khai báo biến
    scanf("%d", &n); nhập giá trị n từ bàn phím
    printf("n = %d", n); // in n ra màn hình
    return 0;
}
Đoạn chương trình trên chạy như sau : B1: khai báo biến n thuộc kiểu số nguyên B2: lấy giá trị của n mà người dùng nhập từ bàn phím gán vào n B3: in ra màn hình " n = {giá trị của n nhận từ bàn phím}" 1.Cú pháp khai báo kiểu dữ liệu cho biến. 
                        
Cú pháp khai báo biến trong C++ như sau:
    kieu_du_lieu ten_bien;

Muốn khai báo nhiều biến cùng lúc và cùng kiểu dữ liệu thì làm như sau:
    kieu_du_lieu ten_bien1, ten_bien2, ten_bien3;
2.Hàm sanf(). 
                        
Hàm scanf() được sử dụng cho đầu vào(nhập giá trị vào). Nó đọc dữ liệu đầu vào từ bàn điều khiển.

Cú pháp : scanf("format string", argument_list); 

    + Format string có thể là %d (số nguyên), %c (ký tự) ,%s (chuỗi), %f (float), v.v

    + Argument_list hiểu đơn giản là biến bạn muốn nhập giá trị vào.
3. Hàm printf() 
                        
Hàm printf() được sử dụng cho đầu ra. Nó in ra lệnh đã cho vào bảng điều khiển (console).

Cú pháp : printf("format string", argument_list);

    + Format string có thể là %d (số nguyên), %c (ký tự) ,%s (chuỗi), %f (float), v.v

    + Argument_list hiểu đơn giản là biến mà bạn muốn in ra màn hình.
